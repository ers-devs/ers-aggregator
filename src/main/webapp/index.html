Entity Registry System (Cassandra and CumulusRDF as backend)

Create new graph

curl -v -X POST -H "Content-Type: application/x-www-form-urlencoded" --data "g=<graph_name>" "http://diufpc304.unifr.ch:8080/ers/graph"

Get all existing graphs

curl -v -X GET -H "http://diufpc304.unifr.ch:8080/ers/graph?g=<graph_name>"

Delete existing graph

curl -v -X DELETE "http://diufpc304.unifr.ch:8080/ers/graph?g=<graph_name>"
